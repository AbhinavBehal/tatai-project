Tatai: Maths Learning Aid

To run this application, JavaFX and HTK are needed and make sure a microphone is connected as default for recording purposes. First, add executable permissions to the 'run.sh' script with this command
    chmod +x run.sh
Then run the script using this command
    ./run.sh

Use:
    To navigate the application, click on the buttons as shown on each page.

    The top right button on the navigation bar allows you to change the current colour theme as well as view statistics.

    You can import your own custom questions in the format of:
        equation=answer
